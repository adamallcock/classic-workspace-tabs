# Chrome Web Store Assets

This directory contains marketing and listing imagery for the Chrome Web Store and GitHub README.

The files are not copied into the packaged extension by `npm run package`.

## Files

- `store-icon-128.png` - 128 x 128 square listing/icon image.
- `marquee-promo-1400x560.png` - 1400 x 560 marquee promotional image.
- `small-promo-440x280.png` - 440 x 280 small promotional image.
- `screenshot-1280x800.png` - 1280 x 800 listing screenshot and README preview image.
